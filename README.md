### 万字文档+实验室耗材管理系统+SSM+JSP+sql文件+软件环境安装包
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、JSP、JavaScript、CSS、echarts可视化

系统功能模块：
本系统包含管理员、教师、学生3个角色。
管理员角色：
1. 教师管理：增删改查教师资料。
2. 学生管理：增删改查学生资料。
3. 实验材料管理：增删改查实验材料信息，实现实验材料信息的批量删除。
4. 老师申请管理：审核老师申请的实验材料信息，统计各位老师申请实验材料的数量，导出文件，在线打印。
5. 学生申请管理：审核学生申请的实验材料信息，统计各位学生申请实验材料的数量，导出文件，在线打印。
6. 耗材统计：统计每学年使用的实验材料总数信息。
教师角色：
1. 实验材料信息：查看实验材料信息，在线申请需要使用的实验材料。
2. 老师申请信息：查看申请的实验材料是否得到管理员审核。
学生角色：
1. 申请材料：申请实验材料需要登记数量，使用内容以及使用日期等资料。
2. 学生申请信息：查看申请的实验材料是否得到管理员审核。
